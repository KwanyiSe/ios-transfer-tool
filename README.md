# iOS Transfer Tool

Cross-platform tool to mimic Android USB transfer mode for iOS.
- Detects iPhone connection & trust
- Auto-creates custom folders (Documents, Downloads, WhatsAppDocs, TelegramDocs, etc.) for user to save files on   phone then be able to access on computer
- Opens Explorer/Nautilus for drag-and-drop transfers
- Runs as background service (like ADB)
(that is it allow user to plug their iphone to a coputer and tranfer files doc etc)
